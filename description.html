Project Description

About the Project

Our project focuses on visualizing the race strategies of Olympic 1500m finalists. Through data analysis and visualization techniques, we aim to uncover insights into the performance tactics employed by elite athletes.

Data Sources

The data used for this project was collected through web scraping techniques, as explained in our "Data" lecture. We obtained split time data for 1500m Olympic finalists and calculated mean speeds per split to analyze race strategies.

Visualization Techniques

Utilizing D3.js, we created interactive line plots to showcase the mean speed per split of athletes. These visualizations allow users to explore different race strategies such as sit and kick, negative split, front running, and even pace.
